Adds a listing of posts using a media file in their content or as a featured image.

== Description ==
This plugin adds a list to the media edit screen of posts using the media file in their content or as a featured image. This list can be helpful if you need to update a photo or attachment that is used in multiple posts on a large site.

== Installation ==
1. If you've downloaded this from the WordPress admin plugin screen then all you'll need to do is click activate!
2. If you've downloaded this from the WordPress plugin repository then drop the file in your 'wp-content/plugins/' directory, then go to the Plugins screen of the admin and activate.

== Frequently Asked Questions ==
= Where does this listing appear? =
On individual 'Edit Media' screens in the admin. Example url: http://samplewordpress.org/wp-admin/post.php?post=10&action=edit
